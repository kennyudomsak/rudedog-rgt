วิเคราะห์เชิงลึก — 6 ข้อสรุปใช้ต่อ

1. งานทำเงิน ฿2.5M / 4 วัน · AOV ฿1,019

เฉลี่ย ฿625k/วัน · บิลใหญ่กว่าค้าปลีกทั่วไป (สตรีทแวร์ขายเป็นชุด)

2. เสื้อยืด = พระเอกกำไร

หมวด TS ฿507k ส่วนลดน้อยมาก · F-โปโล ขาย 1,489 ตัว — สต๊อกให้พอปีหน้า

3. ของแถมน้ำหอม = loss-leader ที่เวิร์ค

ดึงคน 6,641 หัวเข้างาน แต่กำไรน้ำหอมแทบ 0 — โมเดลถูก: ใช้ของถูกล่อ ขายของแพงข้างใน

4. แอด FB ยังเป็นเบอร์ 1 + งานไฮเปอร์โลคอล

FB 40% · นนทบุรี 74% → ปีหน้าเจาะ FB + geo รัศมีนนทบุรี/กทม. ตัดงบ broad

5. RGT พิสูจน์คุ้มค่า

เก็บ 3,001 เบอร์ + เชื่อมยอดขาย (฿833/ลงทะเบียน) + วัดคนเข้าได้ — เครื่องมือนี้ทำซ้ำได้ทุกงาน

6. พีคเสาร์-อาทิตย์ชัด

วัน 4 (อา.) 2,254 หัว · วัน 1-2 (พฤ.-ศ.) เบากว่า → จัดคน/สต๊อก/โปรหนักวันหยุด

⚠️ อ่านก่อนใช้ตัวเลขตัดสินใจ

  • ยอดขาย/บิล/หมวด/ชำระเงิน = เชื่อได้ 100% (export จาก POS โดยตรง)
  • "กำไร ฿2.34M" สูงเกินจริง — POS บันทึก COGS แค่ ฿157k (สินค้าส่วนใหญ่ต้นทุน=0) → กำไรจริงต่ำกว่านี้มาก ต้องใส่ต้นทุนให้ครบก่อนสรุปกำไร
  • คนเข้างาน (RGT) = ใช้ระดับภาพรวม — ยอดรายวัน/ช่องทาง/จังหวัดสอดคล้องงานจริง + conversion สมเหตุสมผล แต่ timestamp ในชีตยังเพี้ยน + เคยมี test ปน (กรองออกแล้ว) เลยไม่เป๊ะ 100%
  • มุมโฆษณายังขาด "ค่าแอดจริง" — ถ้าใส่งบ FB/TikTok ที่ใช้ จะคำนวณ ROAS / ต้นทุนต่อคนเข้างานได้
